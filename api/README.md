# 连接数据库
> 在该根目录下 `.env`文件下填写你对应的sql地址,示例地址如下
```bash
DATABASE_URL="postgresql://postgres:2023@localhost:5432/tally-db?schema=public"
```
> 使用`docker`可参考[链接](https://www.jianshu.com/p/9bb7dfa8d27e)启动或配置数据库
